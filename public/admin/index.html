Stripe

Monitor secure server configuration without exposing credentials.

Secure setup

Stripe secrets cannot be entered or displayed in this admin webpage. Add them as encrypted Cloudflare Pages secrets:

STRIPE_SECRET_KEY=sk_live_...
STRIPE_WEBHOOK_SECRET=whsec_...

Register the webhook endpoint in Stripe as:

https://YOUR-DOMAIN/api/stripe/webhook

Use Stripe test mode first. Never commit keys to GitHub or place them in browser JavaScript.

Release controls and operational checks.

GitHub and Cloudflare Pages

Website code is published through the approved VS Code → GitHub → Cloudflare Pages workflow. Content saved in D1 can publish immediately without changing source files.

  • Use GitHub commits for layouts, code and static content.
  • Use the policy editor for approved policy wording.
  • Keep draft policies private until internal/legal approval.
  • Use Cloudflare deployment previews before structural releases.
